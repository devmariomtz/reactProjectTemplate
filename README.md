## Tech Stack

**Client:** React, TailwindCSS

**Server:** Node


## Run Locally

Clone the project

```bash
  git clone https://github.com/devmariomtz/reactProjectTemplate.git
```

Go to the project directory

```bash
  cd reactProjectTemplate
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
